# 写作风格指南

> **本文件功能**: Phase 4 逐章写作时的风格规范。核心：白话、接地气、日常充实、画面感。

---

## 语言风格

### 对话：像真人说话

**反面**（AI味）:
> "林兄，你的实力确实令人钦佩，但此事恐怕没有那么简单，我建议我们还是从长计议。"

**正面**（白话）:
> "我说老林，你这身手是没话说，但这事没那么简单，别冲动。"

**规则**:
- 短句为主，长句拆开
- 能用一个字不用两个字
- 省略主语（真说话时谁整天"我""你"的）
- 口语化用词（"行""成""没门""少来"）
- 不同身份的人说话方式不同（粗人不说文话，书生不说糙话）

### 叙述：有画面感

**反面**（空洞）:
> 他展现出了强大的实力，令在场的所有人都感到震惊。

**正面**（画面感）:
> 他一拳砸在石墙上，碎石飞溅。旁边蹲着抽烟的两个散工同时站起来，烟都忘了掐。

**规则**:
- 用动作和反应替代形容词
- 用具体数字替代模糊描述（"三米高" > "很高"）
- 用感官替代判断（声音/气味/温度/触感）
- 一个场景最多 3 个感官细节，不要贪多

### 侧面描写：用路人展示世界

**反面**（直白设定）:
> 潜龙城的修行者等级分为九级，主角目前是三级。

**正面**（侧面展示）:
> 街边的告示栏前围了一群人，最新贴出的潜龙榜上，三级觉醒者的名字密密麻麻排了两页，二级的只有十七个。旁边一个卖包子的婶子探着脖子看了看："哟，今年二级的又少了两个。"

**常用侧面手法**:
- 榜单排名（展示主角位置和成长）
- 物价（展示经济体系）
- 路人闲聊（展示世界观和近期事件）
- 摊贩商品（展示科技/修炼水平）
- 建筑和街道（展示城市特点）
- 告示/传单（展示社会秩序和热点）

## 日常充实

### 日常场景的作用

1. **展示世界观**: 通过细节让读者"住"在这个世界
2. **调节节奏**: 不能一直打，日常场景让读者喘口气
3. **侧面描写**: 通过日常中的小细节展示主角的实力变化
4. **伏笔载体**: 日常中不经意提到的东西可能是后面的关键

### 日常场景要素

每 5-8 章安排一段日常，包含至少 2 个：
- 吃饭场景（食物描写 + 对话 + 信息交换）
- 逛街/集市（物价 + 社会风貌 + 偶遇）
- 住宿/休息（居住条件 + 思考/规划 + 梦境回忆）
- 交通/赶路（沿途风景 + 同行对话 + 突发事件）
- 训练/修炼（能力展示 + 难点突破 + 旁人反应）

### 日常对话示例

```
"老板，这碗面多少钱？"
"八铜板。加个蛋两铜板。"
"来碗加蛋的。"他摸出十个铜板放在桌上。
老板瞥了一眼他腰间的令牌："哟，佣兵公会的？面给你多加点。"
```

## 章节结构

### 单章节奏

```
开头（10%）: 承接上文 + 本章场景铺垫
发展（40%）: 核心事件推进，冲突升级
高潮（30%）: 本章最紧张/最燃/最爽的时刻
收尾（20%）: 解决或留悬念 + 章末钩子
```

### 章末钩子类型

| 类型 | 示例 |
|------|------|
| 悬念 | "门被推开的那一刻，他愣住了。" |
| 反转 | "他以为的死局，原来是别人布的棋。" |
| 新信息 | "榜单刷新了，第一名的名字让所有人倒吸一口凉气。" |
| 威胁 | "远处的天空裂开了一条缝，里面有什么东西在往外涌。" |
| 承诺 | "'三天后，我会在擂台上等着你。'" |

### 爽点设计

每 3-5 章必须有一个爽点：
- 打脸（质疑者被啪啪打脸）
- 逆袭（绝境翻盘）
- 获得（得到重要资源/能力/信息/盟友）
- 展示（实力暴露，震惊全场）
- 复仇（对之前的压迫者还击）

## 去AI味核心要点

1. **句式长短交替**: 不要每句都是 15-20 字，偶尔来个 5 字短句
2. **省略过渡**: 不要每段都用"然而""但是""于是""随后"开头
3. **减少修饰**: 一个名词前最多一个形容词，不要"幽暗深邃的古老而神秘的地下洞穴"
4. **对话要有废话**: 真人说话有"嗯""啊""那个"有跑题有废话
5. **场景要有毛边**: 不要完美描述，留点不规整的细节
6. **情绪要克制**: 不要每段都"震惊""难以置信""倒吸一口凉气"
