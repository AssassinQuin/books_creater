# AI味黑名单

> **本文件功能**: AI 写作中常见的套路和用词黑名单。写作时实时对照检查，发现新的 AI 味模式时追加到 Memory。

---

## 禁止的设定套路

| 套路 | 为什么禁止 | 替代方案 |
|------|----------|---------|
| 使用能力的"代价""反噬" | AI最爱编，9成网文都有 | 用资源不足/对手更强/时机不对制造困难 |
| "寿命折损" | 廉价戏剧性 | 直接让后果是具体的（受伤/失去东西/被人追杀） |
| "精神崩溃/失控暴走" | 太常见 | 用具体的情绪反应（发抖/沉默/咬牙/冷笑） |
| "觉醒隐藏血脉" | 毫无铺垫的即兴设定 | 事先埋伏笔，让读者能回溯 |
| "天道/命运选中之人" | 太空 | 用具体事件驱动（被某个人选中/被某件事卷入） |
| "一步登天的机缘" | 跳过成长过程 | 每次升级都有铺垫和代价（不是"代价"设定，是叙事上的努力） |

## 禁止的高频AI词

### 副词类（一个都不要用）

不禁、缓缓、淡淡、微微、轻轻、默默、静静、悄悄、渐渐、悄然、陡然、猛然、蓦然、骤然

### 形容词类（限制使用频率）

深邃、冰冷、炽热、苍茫、浩瀚、恢弘、磅礴、诡异、阴冷、森然、空灵、悠远

### 动词类（找替代）

凝视→盯着/看了一眼 | 感受→觉得/感觉 | 萦绕→绕着/转 | 沉浸→埋头/投入 | 彷徨→犹豫/拿不定主意

### 过渡词（减少使用）

然而、但是、不过、于是、随后、接着、终于、突然、忽然、此刻、刹那间、一瞬间

### 情绪描写（换方式）

不要: "他心中涌起一股莫名的感动"
换成: 用动作表现（攥紧拳头/咬了一下嘴唇/别过头去）

不要: "所有人都被深深震撼了"
换成: 用具体反应（有人退了一步/没人说话/旁边的人互相看了一眼）

## 禁止的句式模式

### 禁止: 过度均匀的信息密度

```
❌ 他走进了房间。房间很大，有五十平方米。墙上挂着一幅画，画的是山水。
   桌上放着一杯茶，茶还冒着热气。窗户开着，微风轻轻吹进来。

✓ 他推门进去。屋里大得有点浪费，就一张桌子一把椅子。桌上那杯茶还冒
   着热气——说明不久前有人来过。窗户开着，风把桌上那张纸吹得哗哗响。
```

### 禁止: 排比句/三连击

```
❌ 他不会放弃，不会屈服，不会退缩。

✓ 他没打算退缩。从来就没有。
```

### 禁止: 过度修饰

```
❌ 他的眼中闪烁着坚定而不屈的光芒，仿佛在黑暗中燃烧的火焰。

✓ 他眯了眯眼。"来吧。"
```

### 禁止: 译制片式对话

```
❌ "我必须告诉你，这件事比你想象的要复杂得多。"

✓ "这事没那么简单。"
```

## AI味检测方法

### 写完每章后自查

1. 搜索以上高频词，出现超过 3 次就改
2. 检查是否每段都是"主语+动词+形容词"的均匀结构
3. 大声朗读对话，如果不像真人说话就改
4. 检查是否有超过 3 句连续的心理描写
5. 检查是否有超过 3 句连续的环境描写
