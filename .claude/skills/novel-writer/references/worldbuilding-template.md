# 世界观建模模板

> **本文件功能**: Phase 2 世界观建模的 6 维度完整模板。每个维度列出必须回答的问题和示例。

---

## 维度 1: 种族与物种

### 必须回答

1. 这个世界有哪些智慧种族？
2. 各种族的生理特征？（体型/寿命/特殊能力）
3. 各种族分布在哪里？
4. 种族之间关系如何？（友好/敌对/从属/贸易/隔绝）
5. 有种族混血吗？混血者的处境如何？

### 示例（废土文）

```
人类: 普通人类，寿命60-80年，分布最广，占据主要城市
变异者: 核辐射后变异的人类，寿命30-50年，体能是普通人的3倍，聚集在辐射区边缘
机械族: 意识上传到机械身体的旧人类，寿命不确定，居住在地下堡垒
关系: 人类视变异者为威胁，机械族封闭不接触，变异者想进入人类城市
```

## 维度 2: 势力与组织

### 必须回答

1. 主要势力有哪些？（至少 3 个）
2. 每个势力的：目标/领导人/地盘/实力等级/手段
3. 势力之间的关系图谱（同盟/敌对/暗中交易/表面和平）
4. 有隐藏势力吗？（暗中操控一切的幕后组织）
5. 势力之间的实力平衡靠什么维持？

### 势力关系示例

```
守望者联盟（政府军）— 控制主城，名义上的秩序维护者
黑市商会 — 控制地下经济，与联盟有秘密协议
拾荒者公会 — 废墟探索者组织，中立但掌握资源信息
净化教团 — 宗教势力，声称能"净化"变异，实际在收集变异者做实验
```

## 维度 3: 地理与区域

### 必须回答

1. 故事发生在多大范围？（城市/国家/大陆/星球）
2. 主要区域及其特色（至少 5 个地点）
3. 每个区域的危险等级
4. 区域之间的距离和交通方式
5. 有禁区/秘境/无人区吗？

### 地图要素

```
起始地: 主角出生/生活的地方（低危险）
核心城: 主城/主势力所在地（相对安全）
练级区: 主角成长过程中去的地方（中等危险）
禁区: 高危险区域，有重要资源或秘密
终极地: 最终决战/真相所在
```

### 距离合理性检查

- 从 A 到 B 步行需多久？
- 有没有快速交通？
- 主角能在合理时间内到达下一个场景吗？

## 维度 4: 能力与等级体系

### 必须回答

1. 力量来源是什么？（修炼/觉醒/系统/血脉/科技/契约）
2. 共有几个等级？每级叫什么？
3. 每级之间的实力差距有多大？（能打几个同级？）
4. 晋升条件（需要什么资源/经历/时间？）
5. 有没有不同体系之间的克制关系？
6. **禁止**: 不准加"代价""反噬""寿命折损"等设定

### 等级表模板

| 等级 | 名称 | 标志性能力 | 晋升条件 | 占比（世界中的比例） |
|------|------|-----------|---------|-------------------|
| 1 | {名} | {能做什么} | {需要什么} | 80% |
| 2 | {名} | {能做什么} | {需要什么} | 15% |
| 3 | {名} | {能做什么} | {需要什么} | 4% |
| ... | ... | ... | ... | ... |

### 升级节奏规划

- 主角多久升一级？（不能太快也不能太慢）
- 番茄文: 每 10-15 章升一级
- 起点文: 每 20-30 章升一级
- 升级必须有足够的铺垫和事件支撑

## 维度 5: 经济与货币

### 必须回答

1. 货币叫什么？有几种面值？
2. 物价体系：
   - 一顿普通饭 = ? 铜币
   - 一把普通武器 = ? 银币
   - 一间普通房子 = ? 金币
   - 修行者需要的高级资源 = ?
3. 普通人月收入多少？修行者怎么赚钱？
4. 贫富差距多大？
5. 有黑市/地下经济吗？

### 物价参考（玄幻模板）

```
1 金币 = 100 银币 = 10000 铜币
一顿饭: 10-50 铜币
普通武器: 5-20 银币
中级灵石: 1 金币
高级丹药: 100-500 金币
普通人月收入: 2-5 银币
```

## 维度 6: 日常生活

### 必须回答

**衣**:
- 不同身份的人穿什么？（材质/颜色/款式有阶级差异吗？）
- 有没有标志性服饰？

**食**:
- 普通人吃什么？修行者吃什么？
- 有什么特色食物/饮品？（可做场景道具）
- 食物来源？（种地/猎杀/采集/交易）

**住**:
- 不同阶层的居住条件？
- 城市布局？（富人区/平民区/贫民窟/商业区）
- 修行者住哪？（洞府/宗门/客栈/野外）

**行**:
- 普通人怎么出行？（走路/马车/公交灵兽）
- 修行者怎么出行？（飞剑/传送/灵兽）
- 从主城到最近的城市要多久？

**社会风俗**:
- 有什么节日/仪式？
- 社会禁忌是什么？（不能提的话题/行为）
- 婚丧嫁娶的规矩？
- 教育体系？（普通人怎么学？修行者怎么学？）

**日常场景素材**（用于侧面描写）:
- 街边摊贩卖什么？
- 酒馆里的人聊什么？
- 城门口的告示贴了什么？
- 集市上的热闹景象？
- 各行各业的从业者状态？

## 维度交叉验证清单

完成所有 6 个维度后，逐项检查：

- [ ] **种族 × 地理**: 各种族的分布与地理设定一致
- [ ] **势力 × 地理**: 各势力的地盘与地理设定匹配
- [ ] **势力 × 经济**: 势力的财力与其行为匹配
- [ ] **能力 × 种族**: 各种族的先天能力差异自洽
- [ ] **经济 × 日常**: 物价与日常生活水平匹配
- [ ] **日常 × 等级**: 不同等级的人日常生活差异合理
- [ ] **时间 × 距离**: 出行时间与距离匹配
