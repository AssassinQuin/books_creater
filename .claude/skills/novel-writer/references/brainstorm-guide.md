# 头脑风暴引导话术库

> **本文件功能**: Phase 0-1 头脑风暴阶段的引导提问模板。AI 一次只问一个问题，根据用户回答选择下一个问题。

## 开场问题（按顺序，从第 1 个开始）

### 1. 画面感

> "你最兴奋的场景是什么？闭上眼想象一下那个画面，描述给我听。哪怕只有一句话也行。"

**追问方向**:
- 如果用户描述了一个战斗场景 → 追问对手是谁、为什么打、主角用了什么能力
- 如果描述了日常场景 → 追问这个场景中什么让主角不开心/不满意（找冲突源）
- 如果描述了感情场景 → 追问两人之间的障碍是什么

### 2. 主角特质

> "主角最让你着迷的特质是什么？用一个词或一句话形容他/她。"

**追问方向**:
- 如果说"聪明" → "什么样的聪明？是善于算计，还是临场反应快？"
- 如果说"狠" → "对谁狠？有没有他不狠的人？为什么？"
- 如果说"善良" → "善良到什么程度？会为陌生人牺牲吗？"
- 如果说"普通" → "那你为什么想写他？他身上一定有什么不普通的地方"

### 3. 读者情绪

> "你希望读者读完最强烈的情绪是什么？"
> - A: 爽（替主角出气/打脸/逆袭）
> - B: 燃（热血沸腾/血脉偾张）
> - C: 虐（心疼/意难平）
> - D: 感动（温暖/治愈）
> - E: 紧张（悬疑/反转）

**追问方向**:
- 选 A → "什么样的爽最让你过瘾？碾压式还是反杀式？"
- 选 B → "最燃的瞬间是什么？突破极限？以一敌百？"
- 选 C → "虐的来源是什么？背叛？失去？求不得？"
- 选 D → "打动你的是什么关系？师徒？兄弟？恋人？"
- 选 E → "你希望反转几次？读者能猜到吗？"

### 4. 对立面

> "有没有一个你特别想写的对手/反派/对立面？他为什么跟主角作对？"

**追问方向**:
- 如果说"没有" → "那主角最大的阻碍是什么？是环境？制度？还是自身？"
- 如果描述了反派 → "他为什么一定要这么做？站在他的角度，他的理由说得通吗？"
- 如果描述了制度/环境 → "这个制度的受益者是谁？谁在维护它？"

### 5. 独特规则

> "这个世界最不一样的地方是什么？什么是这里的人觉得理所当然，但我们会觉得'卧槽'的？"

**追问方向**:
- 如果说"有魔法/修仙" → "这不算独特，几乎所有玄幻都有。**你的**魔法/修仙跟别人的有什么不同？"
- 如果说"没有特别不一样的" → "那我们换个角度：你最讨厌看到什么套路？我们反着来。"

## 深度挖掘问题（根据回答灵活选择）

### 冲突源挖掘

> "主角现在的生活，有什么是他/她无法忍受但又不得不忍受的？"

### 动机挖掘

> "如果主角突然有了无限的实力，他/她第一件事会做什么？这个答案就是他真正的渴望。"

### 世界观种子

> "你最想写的一场战斗/一次相遇/一个秘密是什么？从这个场景往外扩展世界。"

### 节奏偏好

> "你更喜欢快节奏（3 章一个小高潮）还是慢铺垫（10 章一个大高潮）？"

## 头脑风暴结束判断

当以下条件满足时，可以结束头脑风暴：
- ✅ 有至少 1 个明确的核心冲突
- ✅ 主角有基本人设轮廓
- ✅ 读者情绪目标明确
- ✅ 至少有 2-3 个亮点场景

**结束时输出**:
1. 核心冲突候选（1-3 个）
2. 主线方向候选（2-3 个）
3. 亮点场景列表
4. 建议品类和目标平台
5. 让用户选择 → 存入 Memory
