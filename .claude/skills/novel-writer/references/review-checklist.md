# 审阅检查清单

> **本文件功能**: Phase 5 审阅阶段的六维度检查清单。3 个 Agent 并行使用。

---

## Agent A: 逻辑 + 维度一致性

### 时间线检查

- [ ] 本章事件的时间顺序与前后章连贯（无倒挂）
- [ ] 角色移动时间合理（A 地到 B 地的时间与距离匹配）
- [ ] 年龄/成长时间线一致（无"过了三个月但角色没变"）
- [ ] 并发事件不矛盾（A 在甲地时，B 不可能在甲地但故事说 B 不在）

### 空间一致性

- [ ] 角色位置合理（上一章在 A 地，本章突然在 B 地需有交代）
- [ ] 场景描写与已建立的地理设定一致
- [ ] 距离/方向无矛盾

### 能力等级一致性

- [ ] 主角能力等级与当前进度匹配（无突然变强/变弱）
- [ ] 战斗结果与双方等级差合理（低级不能莫名其妙打赢高级）
- [ ] 新技能/新能力有铺垫（无凭空出现）

### 伏笔检查

- [ ] 本章的伏笔已记录到 Memory
- [ ] 已埋伏笔的回收进度是否合理
- [ ] 是否有超过 30 章未回收的伏笔（需要关注）

### 经济一致性

- [ ] 花钱/赚钱与已建立的经济体系一致
- [ ] 物品价格无矛盾（同一东西不能上次 10 银币这次 1 金币，除非有理由）

---

## Agent B: 人设 + 文字质量

### 人物 OOC 检测

- [ ] 主角言行与初始人设一致（无突然性格反转）
- [ ] 配角行为符合已建立的性格
- [ ] 对话风格与人物身份匹配（粗人说粗话，书生说书生话）
- [ ] 人物关系在本章的互动符合当前关系状态

### 去AI味检测

- [ ] 搜索高频AI词（不禁/缓缓/淡淡/微微/...），超过3次标记
- [ ] 检查句式是否过于均匀（每句长度差不多）
- [ ] 检查对话是否有"译制片味"
- [ ] 检查是否有"代价/反噬"等禁止设定
- [ ] 对照 Memory 中的 anti-ai-pattern 黑名单

### 文字规范

- [ ] 错别字/语病
- [ ] 标点错误（对话引号规范）
- [ ] 段落划分（无大段不分段）
- [ ] 无过度环境描写（超过 3 句的连续环境描写）
- [ ] 无过度心理描写（超过 3 句的连续内心独白）

---

## Agent C: 合规 + 竞争力

### 通用合规

- [ ] 无低俗色情/擦边描写
- [ ] 无过度暴力血腥
- [ ] 无封建迷信宣扬
- [ ] 无时政敏感内容
- [ ] 无民族歧视/宗教敏感

### 平台专项（按目标平台）

**番茄专项**:
- [ ] 无标题党/虚假宣传
- [ ] 无诱导互动
- [ ] 无低俗擦边人设/情节

**起点专项**:
- [ ] 无版权问题
- [ ] 无违规导流
- [ ] 符合分类规范

**纵横专项**:
- [ ] 无违规广告植入
- [ ] 符合平台主打品类要求

### 竞争力评估

- [ ] 本章爽点/情绪点是否到位
- [ ] 章末钩子是否有效
- [ ] 是否有同质化严重的内容
- [ ] 差异化亮点是否突出
- [ ] 侧面描写是否充实（日常场景有路人/物价/榜单等细节）

---

## 问题分级标准

| 级别 | 定义 | 处理方式 |
|------|------|---------|
| **致命** | 影响上架/阅读体验（违规、核心逻辑 bug） | 必须立即修改 |
| **严重** | 影响体验（人设 OOC、节奏拖沓、连续 AI 味） | 建议修改 |
| **轻微** | 锦上添花（错别字、用词不当、小优化） | 可选修改 |

## 审阅报告格式

```markdown
# 审阅报告: 第{N}次

**范围**: 第{X}-{Y}章
**日期**: {日期}

## 致命问题 ({N}个)
### 1. {问题标题}
- 章节: 第{N}章
- 位置: 第{M}段
- 类型: {逻辑/合规/人设}
- 描述: {具体问题描述}
- 建议: {可落地的修改方案}

## 严重问题 ({N}个)
{同上格式}

## 轻微问题 ({N}个)
{同上格式}

## 亮点
- {做得好的地方}

## 总评
{一段话总结}
```
