# 品类配置

> **本文件功能**: 各网文品类的特点、爽点模式、节奏偏好。用于头脑风暴阶段建议品类和大纲阶段调整节奏。

---

## 玄幻/仙侠

| 维度 | 内容 |
|------|------|
| 核心爽点 | 升级打怪、碾压、越级战斗、奇遇 |
| 读者期待 | 主角不断变强、探索更大的世界、揭开终极秘密 |
| 节奏 | 番茄: 快升级(10-15章一级)；起点: 稳升级(20-30章一级) |
| 必备元素 | 力量等级体系、功法/技能、灵石/丹药、宗门/势力、秘境/副本 |
| 常见坑 | 等级体系崩坏(后期战力通胀)、地图切换后重复套路 |
| 日常重点 | 坊市淘宝、宗门任务、丹药炼制、功法切磋、排行榜变化 |

## 都市异能

| 维度 | 内容 |
|------|------|
| 核心爽点 | 装逼打脸、身份反差、隐藏实力、逆袭 |
| 读者期待 | 从底层逆袭、身份逐步揭露、都市生活中的超凡体验 |
| 节奏 | 快节奏为主，3-5章一个小高潮 |
| 必备元素 | 隐藏身份/组织、任务/委托、都市地标、金钱/权力 |
| 常见坑 | 战力体系与都市背景冲突、世界观设定经不起推敲 |
| 日常重点 | 职场/校园生活、路边摊/夜市、社交媒体反应、城市氛围 |

## 言情/甜宠

| 维度 | 内容 |
|------|------|
| 核心爽点 | 甜蜜互动、吃醋/占有、身份反差、双向奔赴 |
| 读者期待 | 甜而不腻、虐后更甜、CP 感强 |
| 节奏 | 每 2-3 章一个甜蜜/心动瞬间，偶尔虐 1-2 章 |
| 必备元素 | 误会障碍、情敌/阻碍、共同经历、情感递进 |
| 常见坑 | 误会过久(超过5章不解释)、女主人设弱、男主人设假 |
| 日常重点 | 约会场景、生活细节(做饭/逛街)、朋友圈互动、节日仪式感 |

## 悬疑/推理

| 维度 | 内容 |
|------|------|
| 核心爽点 | 解谜快感、反转震惊、真相大白、智斗 |
| 读者期待 | 线索公平（读者能推理）、反转合理、结局意外 |
| 节奏 | 稳步推进，每 5-8 章一个阶段性真相 |
| 必备元素 | 核心谜题、线索体系、嫌疑人/嫌疑人排除、反转设计 |
| 常见坑 | 线索不公平(读者猜不到)、为反转而反转、结局烂尾 |
| 日常重点 | 调查过程、证据细节、嫌疑人行为、环境氛围 |

## 废土/末日

| 维度 | 内容 |
|------|------|
| 核心爽点 | 生存挑战、资源抢夺、实力碾压、基地建设、揭开真相 |
| 读者期待 | 紧张刺激的生存感、从小到大的势力建设、世界真相 |
| 节奏 | 危机→解决→短暂和平→更大危机，循环升级 |
| 必备元素 | 生存资源(水/食物/能源)、变异/危险、势力/组织、安全区/禁区 |
| 常见坑 | 主角太无敌无生存压力、资源体系前后矛盾、世界观不严谨 |
| 日常重点 | 搜集物资、营地生活、交易/以物易物、幸存者社区、废墟探索 |

## 无限流/游戏

| 维度 | 内容 |
|------|------|
| 核心爽点 | 副本攻略、规则破解、组队/背刺、能力组合 |
| 读者期待 | 有趣的副本设计、智力博弈、团队合作/背叛 |
| 节奏 | 每个副本为一个完整循环（进入→探索→危机→破解→结算） |
| 必备元素 | 副本规则、积分/奖励体系、队友/对手、主系统设定 |
| 常见坑 | 副本设计无新意、规则漏洞、主角开挂过度 |
| 日常重点 | 副本间隙的休息/准备、积分兑换/装备购买、其他玩家互动 |

## 系统流

| 维度 | 内容 |
|------|------|
| 核心爽点 | 系统奖励、任务完成、属性面板提升、抽卡/开箱 |
| 读者期待 | 系统面板的满足感、任务的新奇设计、奖励的惊喜 |
| 节奏 | 任务→完成→奖励→新任务，每 3-5 章一个任务循环 |
| 必备元素 | 系统规则、任务体系、奖励机制、限制/冷却 |
| 常见坑 | 系统太万能(无挑战)、任务重复、系统后期遗忘 |
| 日常重点 | 系统提示音、面板数据变化、任务倒计时、其他人对主角异常的疑惑 |

## 平台 × 品类适配

| 平台 | 最受欢迎品类 | 节奏要求 |
|------|------------|---------|
| 番茄 | 都市异能、甜宠、系统流 | 最快，开篇即爽 |
| 起点 | 玄幻、仙侠、悬疑 | 中等，允许铺垫 |
| 纵横 | 玄幻、仙侠 | 中等，传统节奏 |
